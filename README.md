# ONE MORE BLOCK

Pygame로 만든 스태킹 게임.

## Controls
- Left Click / SPACE: Drop
- SPACE / ENTER (Game Over): Restart
- F11: window size toggle (windowed_max ↔ windowed)
- B: BGM ON/OFF
- [ / ]: BGM volume down/up
- LEFT / RIGHT (Game Over): Skin select

## Skins (Themes)
- Sky: 기본 (눈부심 완화 톤 적용)
- Neon: BEST 25 달성 시 해금
- Paper: PERFECT 누적 10회 달성 시 해금

## PERFECT Feedback
- 카메라 흔들림 + 파티클 + 클릭 SFX
