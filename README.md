# ONE MORE BLOCK

- 착지 시 **겹친 부분만 남는(트림)** 방식으로 점점 더 좁아지는 스택을 쌓는 게임# ONE MORE BLOCK

"한 번만 더…" 하다가 계속 하게 만드는, 타이밍 기반 블록 트리밍 게임.

## 게임 핵심
- **클릭/스페이스**로 타이밍 맞춰 블록을 떨어뜨리고,
- 착지 시 **겹친 부분만 남는(트림)** 방식으로 점점 더 좁아지는 스택을 쌓는 게임.  (실패하면 종료)

겹침이 적을수록 블록이 더 좁아지고,
겹침이 충분히 없으면 바로 게임 오버!

---

## 조작
- **좌클릭 / Space**: 블록 떨어뜨리기(DROP)
- **F11**: 창 모드 <-> 테두리 없는 최대화 전환(풀스크린 아님)
- **ESC**: 종료
- **게임 오버 후**: 클릭/Space로 다시 시작

---

## 점수(HEIGHT)
- 블록을 **성공적으로 착지**시킬 때마다 HEIGHT +1
- 최고 기록(BEST)은 `save.json`으로 저장되어 다음 실행에도 유지됨

---

## PERFECT & 콤보 보상
- 겹침 비율이 일정 이상이면 **PERFECT** 판정
- PERFECT가 연속으로 쌓이면 **콤보**가 증가
- 일정 콤보마다 **다음 블록 폭 보너스**가 1회 적용됨
  - (너무 빡세게 좁아지는 걸 한 번 완화해주는 “보상”)

---

## 잘린 조각(Shard) 연출
- 트림으로 잘린 왼쪽/오른쪽 조각이 **떨어지는 연출**로 생성됨
- 물리 정확도보다는 “보기 좋게” 떨어지도록 단순 중력 적용

---

## 손맛(연출)
- 착지 순간:
  - 블록이 살짝 눌리는 **스쿼시 효과**
  - 화면이 살짝 흔들리는 **쉐이크 효과**
- PERFECT면 쉐이크가 더 강하게 들어가서 “쾌감”이 강조됨

---

## 난이도 곡선(시간이 지날수록 어려워짐)
HEIGHT가 올라갈수록:
- 좌우 이동 속도 증가
- 낙하 속도 증가
- 블록 폭 랜덤 변화 폭 증가
- 스폰 위치 랜덤 오프셋 증가

(초반은 안정적, 후반은 손이 바빠지는 구조)
